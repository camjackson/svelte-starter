# svelte-app

Starter repo for a Svelte app. Features:

- Svelte
- TypeScript
- Tailwind CSS
- Jest
- Testing Library
- Prettier
- Eslint

Linting is limited to .ts files (not .svelte files), pending TypeScript support in
[eslint-plugin-svelte3](https://github.com/sveltejs/eslint-plugin-svelte3/issues/68).
